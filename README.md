# snowmix-aur-pkg
custom snowmix AUR package

Arch Linux AUR package.

**Install instructions**
```
git clone https://github.com/nt74/snowmix-aur-pkg.git
cd snowmix-aur-pkg/
makepkg -s
sudo pacman -U snowmix-0.5.1.1-1-x86_64.pkg.tar.zst
```
